# Design Direction

## Theme

Neo-minimal, dark, and quietly luminous with a soft green hue.

## Palette

Use one central green family for emphasis and state:

- `#F0FDF4`
- `#DCFCE7`
- `#BBF7D0`
- `#86EFAC`
- `#4ADE80`
- `#22C55E`
- `#16A34A`
- `#15803D`
- `#166534`
- `#14532D`

Support it with deep slate neutrals for structure and contrast.

## Visual Rules

- Start with layout, spacing, and hierarchy.
- Use color sparingly for action, focus, selected states, and active data.
- Prefer soft radius and subtle elevation.
- Avoid loud gradients unless they serve the hero moment.
- Keep animations restrained and purposeful.
- Ensure text remains readable over all surfaces.

## Application Usage

- The landing page should feel like a calm product introduction, not a flashy promo site.
- The visualizer should still feel energetic, but the accent color should read as green-first rather than cyan-first.
- Controls and interactive states should reuse the shared mint/green accent system.

## Token Direction

Recommended semantic split:

- background surfaces: slate/ink
- primary accent: mint/green
- positive state: emerald
- warning state: amber
- destructive state: red

## Notes

The design should feel technical, composed, and a little futuristic, but never noisy.
